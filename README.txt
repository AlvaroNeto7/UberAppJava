===============================
 UberApp - Controle de Ganhos
===============================

Este aplicativo em Java permite que motoristas de aplicativos como Uber
registrem seus dias de trabalho, calculando ganhos, despesas e médias semanaismensais.

-------------------------------
🔧 REQUISITOS
-------------------------------
- Java JDK instalado (versão 8 ou superior)
- Terminal ou Prompt de Comando
- Editor de texto (Notepad, VS Code, etc.)

-------------------------------
📁 ARQUIVOS INCLUÍDOS
-------------------------------
- DiaTrabalho.java
- RelatorioFinanceiro.java
- AppMotorista.java

-------------------------------
▶️ COMO USAR
-------------------------------

1. Abra o terminal (cmd, PowerShell ou Terminal LinuxMac)

2. Navegue até a pasta onde estão os arquivos
   Exemplo (Windows)
   cd CCaminhoParaUberAppJava

3. Compile o programa com
   javac DiaTrabalho.java RelatorioFinanceiro.java AppMotorista.java

4. Execute o programa com
   java AppMotorista

-------------------------------
📊 FUNCIONALIDADES
-------------------------------
✔ Registrar horas trabalhadas, ganhos e gastos de um dia
✔ Salvar dados automaticamente em arquivo CSV (ganhos.csv)
✔ Calcular médias financeiras com base nos dias registrados

-------------------------------
💡 DICAS
-------------------------------
- O arquivo ganhos.csv será criado automaticamente.
- Você pode abrir o CSV com Excel ou Google Sheets para visualização.
